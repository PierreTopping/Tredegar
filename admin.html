📝 Archiving Guidelines

  • Images: Clean up dust/scratches if possible, crop borders tightly, and save files as lowercase `.jpg`.
  • Maps: File name structure must start with `Map_` followed by description (e.g., `Map_Old4.jpg`).
  • Videos: Ensure any video file added is web-optimized for fast playback on mobile networks.
